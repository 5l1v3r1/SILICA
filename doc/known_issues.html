SILICA is an experimental software, and here are some known issues about SILICA 0.3:

  • Incompatable with scim when compiled with the GTK+ 2 widgets, due to some ABI conflicts.
  • Sometimes crashes and gives the error message "glibc detected : double free or corruption (fasttop)". To fix this error export the environment variable MALLOC_CHECK_=0.

If you found any bug using SILICA, please contact us.
