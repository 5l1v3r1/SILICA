Install Binary Packages

Restricted by our ability, currently we are not able to provide any other package but tarball. We will pack SILICA in .deb, .rpm in the future.

Download

Packages can be downloaded from http://sourceforge.net/projects/silica/

Install/Remove

tarball

First, extract the files from the tarball by typing 

    tar -xvf silica-binary.tar

in the terminal.

Then, cd to silica-binary, type 

    sudo make install

to install SILICA to your machine, or type 

    sudo make uninstall

to remove SILICA from your machine.


Compile from Source

SILICA is a free & open source software. You can get the source and compile it by yourself.
NOTE : to compile silica, you need to install Lazarus 0.9.26-4 or later version.

Together with binary packages, we provide the source on SourceForge as well. You can download the source of the latest stable version from :
http://sourceforge.net/projects/silica/files/SILICA/.

Extract the files from the tarball, and type 

    make all

to compile. When the compilation is done, type 

    sudo make install

to install SILICA to your machine, or type 

    sudo make uninstall

to remove SILICA from your machine.

HINT: SILICA uses SVN to manage the source. You can get the latest unstable version from our SVN repository: https://silica.svn.sourceforge.net/svnroot/silica
WARNING : the source in SVN is not stable, and may harm your system.
